## Запуск контейнера 
- Внимательно прочитать .env.example
- Создать .env.prod
- запустить одной командой из корня проекта

##### Команда запуска
```bash
docker-compose up --build 
```
###### Больше ничего не нужно делать, миграции будут применены база данных будет заполнена фикстурой сама
